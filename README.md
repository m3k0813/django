# 디장고(django) 가상환경 셋팅

<pre>
<code>
python -m venv 가상환경이름
</code>
</pre>
- 가상환경 만들기

<pre>
<code>
가상환경이름/Scripts/activate
</code>
</pre>
- 가상환경 실행

<pre>
<code>
deactivate
</code>
</pre>
- 가상환경 끄기

<pre>
<code>
django-admin startproject project 이름
</code>
</pre>
- Project 만들기

<pre>
<code>
python manage.py runserver
</code>
</pre>
- 서버 실행 시키기 주소는 http://127.0.1:8000/ 

<pre>
<code>
python manage.py startapp 앱 이름
</code>
</pre>
- 프로젝트의 구성 단위인 앱 만들기
